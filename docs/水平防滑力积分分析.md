## 1. 背景与目标

此前的防滑工作（方案一 contact-center、方案二 vertical-support S）都聚焦于
**竖直方向**（抗重力抬升）的滑移检测。本工作补充**水平面内**的滑移分析：抓取物体
在水平面内的相对滑动/漂移同样会导致抓取失败，尤其在低摩擦、侧向载荷或水平搬运时。

目标：定义一个**与手部姿态绑定的水平坐标系**，把所有手→物接触力（法向 + 切向）
分解到该坐标系的 X、Y 两个水平轴上并做积分，观察不同摩擦条件下水平力/冲量与物体
水平漂移的关系。

---

## 2. 水平坐标系定义（用户提案）

在世界水平面（垂直于重力）内定义：

| 轴 | 定义 | 物理含义 |
|----|------|----------|
| **X̂** | 四个手指（食/中/无名/小指）平展方向 | 沿手指伸展方向 |
| **Ŷ** | 与 X̂ 垂直、指向右侧：`Ŷ = X̂ × ẑ_up` | 手指侧向（向右） |
| ẑ_up | 抗重力方向（竖直，另由方案二处理） | 抓握/抬升方向 |

**X̂ 的计算**：取四指的 knuckle→tip 向量（`right_hand_*_link → right_*_tip`）求平均，
去掉竖直分量后归一化，得到水平面内的"四指平展方向"。**Ŷ = X̂ × ẑ_up** 保证在水平面
内、指向四指的右侧。坐标系逐帧动态计算（随手部姿态更新），在以竖直抬升为主的 extend
段基本稳定。

实测抓取完成时（番茄酱右手）：

```
X̂ ≈ [ 0.92, -0.38, 0.00]   (四指平展方向, 水平)
Ŷ ≈ [-0.38, -0.92, 0.00]   (垂直向右, 水平)
```

代码：`src/sim/slip_horizontal.py::compute_hand_horizontal_frame`

## 3. 力积分定义

对每个手→物接触点，将接触力分解为世界系下的法向分量 `f_n` 与切向分量 `f_t`
（`f = f_n + f_t`），随后把**完整力**（含法向 + 切向）投影到 X̂、Ŷ 并对所有接触点求和：

$$
F_x(t) = \sum_{c} \mathbf{f}_c \cdot \hat{X}, \qquad
F_y(t) = \sum_{c} \mathbf{f}_c \cdot \hat{Y}
$$

其中求和覆盖当前时刻所有手→物接触（即"所有力"的空间积分 = 净水平力）。

时间积分（冲量）：

$$
J_x = \int F_x\,dt, \quad J_y = \int F_y\,dt, \quad
J_{|h|} = \int \sqrt{F_x^2 + F_y^2}\,dt
$$

同时单独记录法向投影 `Fx_normal/Fy_normal` 与切向投影 `Fx_tangent/Fy_tangent` 用于诊断。

代码：`src/sim/slip_horizontal.py::measure_horizontal_forces` /
`HorizontalImpulseIntegrator`

## 4. 实验设置

- 场景：SPIDER arcticv2 番茄酱右手抓取（`s01-ketchup_use_01`）
- 流程：抓取轨迹（0–3s）→ extend 段（3–5s，模仿末 1s 手指控制 + 腕部竖直抬升 10cm）
- 摩擦条件：friction ÷2（×0.5）、÷4（×0.25）、÷8（×0.125）
- 采样：`sim_dt = 0.01s`，全程记录 X/Y 力与积分

运行：

```bash
python3 scripts/build_spider_ketchup_right.py    # 需先 bash scripts/setup_spider.sh
python3 scripts/plot_horizontal_force_curves.py
```

## 5. 结果

### 5.1 关键指标（全程）

| 摩擦 | 轨迹接触步 | extend 接触步 | ∫Fx [N·s] | ∫Fy [N·s] | ∫\|F_h\| [N·s] | 水平漂移 [cm] | Δz [cm] |
|------|-----------|--------------|-----------|-----------|---------------|--------------|---------|
| ÷2 | 136 | 99 | 333.3 | 225.1 | **616.8** | 5.8 | −18.7 |
| ÷4 | 28 | 0 | 12.4 | 3.5 | **18.5** | 18.5 | −18.7 |
| ÷8 | 14 | 0 | 2.4 | −1.0 | **3.0** | 24.2 | −14.0 |

### 5.2 观察

1. **摩擦越低，累计水平力冲量越小、物体水平漂移越大**：
   - ÷2：抓取保持到 extend 段（235 接触步），水平力冲量最大（617 N·s），
     物体被"抓住"，水平漂移最小（5.8cm），最终在抬升中竖直滑落（−18.7cm）。
   - ÷4：仅在抓取轨迹段有 28 接触步，抬升前物体已从手中滑出，冲量骤降到 18.5 N·s，
     物体水平漂移 18.5cm（滑到一侧落地）。
   - ÷8：更早失稳（14 接触步），冲量仅 3 N·s，水平漂移最大 24.2cm。

2. **水平力冲量是抓取水平稳定性的敏感指标**：从 617 → 18.5 → 3.0 N·s（约 30× / 200× 衰减），
   比单纯的接触数或竖直位移更能区分"抓得住 / 抓不住"。

3. **法向 vs 切向**：见 `horizontal_normal_tangent.png`，抓取稳定时 X/Y 净力主要来自
   多指法向力在水平面的合成（对握合力），切向力提供额外的水平约束；失稳后两者同时归零。

## 6. 讨论与后续

- **本工作是"分析/度量"层**：给出手部水平坐标系下的力积分信号，尚未闭环控制。
- **与竖直方案的互补**：竖直方案二用 `S = Σ f·ĝ` 检测抗重力支撑衰减；本工作用
  `Fx, Fy` 及其冲量刻画水平面内的净力与约束。两者可组成完整的 3D 抓握力监控。
- **潜在水平防滑触发量**：
  - 净水平力方向突变（物体被推向某侧）；
  - `∫|F_h| dt` 增速下降（接触约束丢失）；
  - 水平漂移速度 `d|Δ_horiz|/dt` 超阈值。
- **后续**：
  1. 在 ÷2 边界场景下，把水平力信号接入 `GripBoostController`，验证能否抑制水平漂移；
  2. 将 X/Y 力积分纳入 NN 特征（见[方案一二神经网络防滑检测设计](./方案一二神经网络防滑检测设计.md)）；
  3. 真机 XHAND 上用 taxel 切向力阵列复现 Fx/Fy 分解。
