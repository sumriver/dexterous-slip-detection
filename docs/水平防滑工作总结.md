## 1. 问题说明

### 1.1 背景
此前的防滑工作聚焦**竖直方向**（抗重力抬升）：方案一（接触中心发散 center divergence）与方案二（垂直支撑力 S 及其 S/S_avg 比值）。但抓取失败也可能发生在**水平面内**——物体相对手指的横向滑移/漂移，尤其在低摩擦、侧向载荷或水平搬运时。

### 1.2 本轮任务
在水平面内建立一套**与手部姿态绑定**的力分析框架，考察不同摩擦条件下水平接触力的行为，并探索水平防滑的检测量与缓解策略。

### 1.3 核心澄清（问题的真正落点）
讨论过程中明确了真正的目标：**如何"提前"知道要滑动（早期滑移检测/预测）**。
缓解手段是**下游、可插拔**的，有很多种（增握力、双手协同、让瓶子平躺、降低加速度……）；
而科学核心是**预警**——在物体宏观滑落之前就拿到信号。据此，"S 下降 / 脱手 / 大幅漂移"都属于**滞后**信号，需要更靠前的量。

## 2. 假设

| 编号 | 假设 |
|------|------|
| **H1（坐标系）** | 可用手部锚定的水平坐标系描述接触力：**X̂ = 四指平展方向**（四指 knuckle→tip 均值的水平投影），**Ŷ = X̂ × ẑ_up**（垂直、指向右侧）。 |
| **H2（平衡判据）** | 稳定不滑时，水平**净合力 ≈ 0**（多指对握力相互抵消）；发生滑动时净合力 ≠ 0。 |
| **H3（法向/切向分解）** | 稳定时水平**法向净力**与**切向净力**等大反向、相互抵消；摩擦不足时切向"跟不上"法向，二者出现缺口。 |
| **H4（慢速抓取）** | 减缓抓取速度可减少"快速挤压把物体挤飞"的动态失稳。 |
| **H5（松握缓解挤出）** | 当水平 `\|F法向\| > \|F切向\|`（摩擦无法平衡法向内推）时，**减小握力**可减轻横向挤出。 |
| **H6（早期检测，reframe）** | **摩擦锥利用率** `ρ = \|f_t\|/\|f_n\| → μ` 与**初始滑移（partial slip）**能在物体宏观运动前预警，提前于 S 下降等滞后量。 |

## 3. 验证

### 3.0 关键修正：接触力坐标系 bug（前置条件）
分析"净合力是否为零"时发现 `decompose_contact_force_on_object` 的接触系变换错误：MuJoCo `contact.frame` **行主序**（行=接触轴），转世界系应为 `frame.T @ v`、法向取 `frame[0,:]`，原代码用了 `frame @ v` / `frame[:,0]`，导致**对握力无法相互抵消**。

牛顿验证（准静态净接触力应 ≈ `[·,·,+mg]`，mg≈4.7N）：

| | 净接触力（世界系） |
|--|------------------|
| 修复前 `frame @` | `[-14, 185, -329]` N ❌ |
| 修复后 `frame.T @` | `[-0.9, -1.2, 6.5]` N ✅ |

> 竖直方案二用 S/S_avg **比值**，对缩放不敏感，修复后 `friction_div2 --antislip` 仍 **PASS（+8.7cm, 200/200）**；但水平净合力分析必须用修复后的力才成立。

### 3.1 实验一：X/Y 力积分（验证 H1、H2 的积分侧）
| 摩擦 | ∫Fx | ∫Fy | ∫\|F_h\| [N·s] | 水平漂移 [cm] |
|------|-----|-----|---------------|--------------|
| ÷2 | 0.14 | −0.38 | 5.68 | 5.8 |
| ÷4 | 0.35 | −0.13 | 1.43 | 18.5 |
| ÷8 | 0.34 | −0.30 | 0.94 | 24.2 |

- **带符号 ∫Fx、∫Fy ≈ 0** → 抓取中水平净合力基本抵消（支持 H2）。
- **∫\|F_h\| 随摩擦单调下降**（接触维持时长×强度），水平漂移单调上升。

### 3.2 实验二：净合力"绝对值 + 方向"（验证 H2，加 baseline + 准静态保持段）
| 场景 | 阶段 | 净 \|F_h\| [N] | 不平衡比 r=\|ΣF\|/Σ\|F\| |
|------|------|--------------|------------------------|
| baseline（不滑） | hold | **1.35** | **0.0013** |
| baseline | lift | 1.24 | 0.0014 |
| ÷2 | hold | 0.69 | 0.042 |

- **稳定抓取：净水平合力 ≈ 1N、不平衡比 ≈ 0.001**（对握力几乎完全抵消）→ **H2 验证成立**。
- 即便竖直抬升，水平净合力仍≈0（抬升是竖直的，水平不加速）。
- **÷2 抬升临界（t≈4s）不平衡比瞬时跳升到 0.32**（滑移起始信号），随后脱手。

### 3.3 法向净力 / 切向净力分解（验证 H3）
| 场景 | 法向净 ⟨\|F_n\|⟩ | 切向净 ⟨\|F_t\|⟩ | 切/法比 | 合力 ⟨\|F_h\|⟩ |
|------|----------------|----------------|--------|---------------|
| baseline | 66.1 N | 65.7 N | 0.993 | 1.70 N |
| ÷2 | 119.3 N | 118.7 N | 0.995 | 2.46 N |
| ÷4 | 48.7 N | 44.6 N | 0.917 | 5.10 N |
| ÷8 | 20.4 N | 14.2 N | 0.693 | 6.70 N |

- 法向净与切向净各自都很大，但**等大反向、相互抵消** → 总净力小（支持 H3）。这是水平抓握平衡条件 `F_tangent ≈ −F_normal`。
- **切/法比随摩擦下降**（0.99→0.92→0.69）：摩擦越低，切向越"跟不上"法向 → **摩擦饱和的直接征兆**（引出 H6）。

### 3.4 实验三：÷4 慢速抓取 + 松握（验证 H4、H5）
三组对比（÷4，起始 z≈18.9cm）：

| 配置 | 轨迹末 z | 保持末 z | 结束 z | 水平漂移 | 接触步 |
|------|---------|---------|--------|---------|-------|
| A 正常速+开环 | 0.1cm | 0.1cm | 0.1cm | 18.5cm | 28 |
| B 慢速+开环 | **21.1cm** | 0.2cm | 0.2cm | 65.8cm | 306 |
| C 慢速+松握 | **21.2cm** | 0.2cm | 0.3cm | **42.7cm** | 320 |

- **H4 成立且为主导改善**：A（正常速）抓取瞬间脱手；B/C（慢速）稳稳撑过整个抓取轨迹。
- **H5 有正向但有限效果**：C 相比 B 水平漂移 65.8→42.7cm、接触更久；松握只短暂触发。
- **÷4 无法静态保持**：B/C 都在保持段滑脱；额外叠加"竖直增握"也仍失败——说明 ÷4 已超出纯握力调节的能力。

对比视频：`data/horizontal_grip_release/grip_release_div4_A_vs_C.mp4`（左 A=Fail，右 C=Success，抓取阶段）。

### 3.5 分阶段策略尝试（横向 vs 竖直的张力）
尝试"慢速抓取 + 抓取期松握 + 保持/抬升期竖直增握（方案二）"，÷4 仍在保持段脱手。
根因：**横向防挤出要"松握"，竖直抗下坠要"增握"，两者方向相反**，÷4 摩擦太低无法两全。

## 4. 结论

1. **坐标系 & 平衡判据成立（H1、H2）**：手部锚定的 X/Y 框架下，稳定抓取水平净合力≈1N、不平衡比≈0.001；滑移起始时不平衡比跳升（÷2 见 0.32）。
2. **法向/切向分解揭示物理本质（H3）**：法向净与切向净等大反向抵消；**切/法比**随摩擦下降（0.99→0.69），是摩擦饱和的直接指标。
3. **慢速抓取是最有效的单项改善（H4）**：显著改善抓取阶段成功率（÷4 从"瞬间脱手"到"稳稳握住"）。
4. **松握能减轻横向挤出但有限（H5）**：漂移、接触保持有改善，但不能替代竖直支撑。
5. **横向与竖直目标相反**：防挤出松握 vs 抗下坠增握；÷4 已超出纯握力调节边界。
6. **问题的正确落点是"早期检测"**：上面很多量（S 下降、脱手、大漂移）都是**滞后**的；**切/法比 / 摩擦锥利用率**才指向"提前知道"。
7. **修复了一个影响面较广的接触力坐标系 bug**（`frame.T`），使净力/平衡类分析成立。

## 5. 后续推荐

### 5.1 早期滑移预警框架（核心方向）
按"能提前多久"分层，越靠前越早：

| Tier | 信号 | 提前量 | µ 依赖 |
|------|------|--------|--------|
| **T1 摩擦锥裕度** | `ρ = \|f_t\|/\|f_n\|`，风险 `ρ/μ→1` | 物体动之前 | µ 已知最佳 |
| **T2 初始滑移/微观** | 接触中心 COP 漂移、切向力饱和、部分滑移（taxel 斑块平移）、微振动 | 滑动刚萌芽 | µ 无关（真机首选） |
| **T3 宏观确认** | 物体相对速度、S/S_avg 下降、不平衡比跳升 | 已在滑（滞后） | — |

### 5.2 建议的验证实验（量化"提前量"）
1. 逐接触算 `ρ=|f_t|/|f_n|`，聚合成摩擦利用率曲线；
2. 定义预警触发（`ρ/μ` 超阈；µ 未知时用 COP 漂移速率 / 部分滑移比例）；
3. 定义"真滑移时刻"（物体相对手滑移速度超阈 / z 起始下降）；
4. 报告 **lead time = 真滑移时刻 − 预警时刻**，比较 ÷2/÷4/÷8 各信号的提前量。

### 5.3 场景扩展
- 当前 ÷2/÷4/÷8 是**竖直抬升主导**（滑移以脱手下坠收场）；增加**侧向剪切 / 水平搬运**场景，可让水平净力/方向信号充分发展、验证方向锁定。

### 5.4 缓解策略（与检测解耦，可插拔）
- 增握（竖直方案二）、松握（横向防挤出）、**双手协同**、**改变物体位姿（如平躺降低力臂/增大接触）**、降低运动加速度；
- 由早期预警统一触发，按场景选择或组合。

### 5.5 学习型与真机
- 将 X/Y 净力、切/法比、COP 漂移、`ρ/μ` 纳入 NN 特征（见[方案一二神经网络防滑检测设计](./方案一二神经网络防滑检测设计.md)）；
- 真机 XHAND：用 12×10×3 taxel 阵列复现切向/法向分解、COP 漂移与部分滑移检测（µ 无关信号为主）。
