**⚠️ 拇指几何硬约束（数学证明）：**

通过 MuJoCo 模型坐标系下的指尖位置计算，**拇指无法真正对握**：

```
拇指指尖局部坐标：(+0.083, 0, +0.078) — 位于 +X 半空间
食指指尖局部坐标：(+0.033, 0, +0.165) — 位于 +X 半空间
对立角（opposition angle）：35.7°（不变量）
```

**结论**：任何旋转+平移都无法让拇指与食指位于世界坐标系的相反侧。
这意味着 **Shadow Hand 无法从真正对侧包住圆柱形物体（如瓶子）**。这是物理层面的不可克服限制，任何算法都无法解决。

## 4. 能量流方法（NAIST+Honda）核心原理

### 4.1 论文信息

### 4.2 核心思想：虚拟能量容器

**将抓取的物体抽象为一个"虚拟能量容器"**，通过监测能量是否守恒来判断是否发生滑动。

#### 能量状态定义

系统状态 x_t ∈ R^10：

```
x_t = [P^A, P̃^R, Θ]
      ├── P^A ∈ R^5：各手指施加功率（接触力 × 接触点速度）
      ├── P̃^R ∈ R^3：无质量假设下保留在抓取中心的功率
      └── Θ ∈ R^3：抓取中心朝向（四元数表示）
```

#### 施加能量 E^A（Applied Power）

对所有手指接触点积分：

```
P^A_i = F_i · v_i    （第 i 个手指的接触力 × 接触点速度）
P^A = [P^A_1, ..., P^A_5]
```

#### 保留能量 P̃^R（Retained Power under massless assumption）

假设物体无质量时，抓取中心应保留的功率：

```
P̃^R = f(ΣF_i, v_center, ω_center)
```

#### 质量估计 m̃_t

```
m̃_t = 1^T · P^A / 1^T · P̃^R    （Eq.7）
```

**物理含义**：在恒定质量假设下，施加功率与保留功率的比值应稳定。**如果发生滑动，能量会"泄漏"，质量估计值会剧烈波动**。

#### 滑动检测 = 质量估计波动

```
滑动判据：|m̃_t - median(m̃_t)| > threshold
```

### 4.3 在线学习：LGM-FF

**LGM-FF（Fourier-featured Linear Gaussian Model）** 是核心学习引擎：

- **概率动力学模型**：学习状态转移 p(x_{t+1} | x_t, u_t)
- **在线增量学习**：每次 trial 实时采集 (x_t, u_t, x_{t+1}) 三元组，trial 结束后更新模型
- **Fourier Features**：用傅里叶特征扩展线性模型的非线性表达能力
- **收敛速度**：仿真 ~15 trials / 1200+ samples / 2 分钟；真机 8 trials / ~87s

### 4.4 力优化：pMPC（probabilistic MPC）

- **控制频率**：10 Hz（底层仿真 500 Hz）
- **力约束窗口**：±3 N（围绕当前力值）
- **力限**：[0, 30] N（仿真）/ [0, 20] N（真机）
- **损失函数**：ℓ(x_k) = -exp(-α|m̃_k - median(m̃_t)|)，α=200
