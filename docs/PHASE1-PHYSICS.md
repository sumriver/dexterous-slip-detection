# Phase 1 物理仿真说明

## 已删除（假性成功来源）

| 组件 | 问题 |
|------|------|
| `GraspCoupler` | 每帧把瓶子 qpos 写死在前臂坐标系，违反重力 |
| 手指 qpos 运动学覆写 | 绕过接触力学 |
| LIFT 阶段零接触强制锁瓶 | 腕部"吸瓶" |
| 仅看 bottle_z / tilt 判 PASS | 不验证接触承重 |

## 当前原则

1. **瓶子**：只有重力 + 接触力 + 摩擦决定运动
2. **手指**：仅通过 `data.ctrl` 力矩驱动（动态闭合）
3. **手腕轨迹**：规划路径 + 限速移动（非吸盘）
4. **抬升/翻转**：仅当 HOLD 末尾检测到 ≥3 个手–瓶接触且瓶子未倾倒才启用
5. **PASS**：仿真未爆炸 + 接触数 + 瓶子实际抬升（无绑瓶）

## 已知硬约束

Shadow Hand 对**竖立圆柱瓶**：

- 拇指无法对握（对立角 ~35.7°，见研究报告）
- 物理抓取时瓶子易倾倒、穿模、飞出
- **当前诚实结果：grasp / lift / flip 均为 FAIL**

建议 Phase 1 验证改用 **方块物体 + 三指侧捏**（与 NAIST+Honda 论文一致），
圆柱瓶场景留到 XHAND1 真机或换 Allegro/LEAP 手模之后。
